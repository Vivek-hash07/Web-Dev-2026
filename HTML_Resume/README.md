# HTML Resume

A professional resume created using HTML. This project showcases my skills, experience, and education in a clean, web-based format.

## Overview

This repository contains an HTML-based resume that displays my professional information, work experience, technical skills, and educational background in a HTML based website.

## Features

- **Clean Design**: Professional and easy-to-read layout
- **Semantic HTML**: Proper HTML structure for accessibility

## Files

- `index.html` - Main resume file
- `readme.md` - This file

## How to Use

1. Open `index.html` in a web browser to view the resume
2. The resume displays:
   - Personal information and contact details
   - Professional summary
   - Work experience
   - Technical skills
   - Education

## Customization

To customize this resume:
1. Edit the HTML content in `index.html` with your personal information
3. Save your changes and refresh the browser to see updates

- HTML5
