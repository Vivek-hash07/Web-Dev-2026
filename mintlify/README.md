# Mintlify — Clone README

A static recreation of the **Mintlify** marketing site (“The Intelligent Document Platform”).

---

## Sections recreated

1. **Navigation bar** — Logo, nav links (Resources, Documentation, Customers, Blog, Pricing), “Contact sales” and “Start for free” buttons.
2. **Hero** — “NEW” pill (self-updating docs), main headline “The Intelligent Knowledge Platform”, subtitle, email input + “Start now” CTA.
3. **Overview image** — Full-width product/dashboard screenshot.
4. **Brand logos** — Grayscale customer/partner logo strip.
5. **Built for the intelligence age** — Section heading and two feature cards:
   - **LLMS.TXT & MCP** — “Built for both people and AI” with supporting copy and image.
   - **AGENT** — “Self-updating knowledge management” with copy and image.
6. **Assistant** — “ASSISTANT” label, “Intelligent assistance for your users” heading, description, and screenshot.
7. **Enterprise** — “ENTERPRISE-REINVENTION” label, “Bring intelligence to enterprise knowledge” headline, “Explore for enterprise” CTA, and two columns:
   - Build with partnership  
   - Compliance and access control  
   Plus branding/screenshots.
8. **Demo / CTA** — “Make documentation your winning advantage”, “Get started for free” and “Get a demo” buttons, and two cards: “Pricing on your terms” and “Start building” (Quickstart).
9. **Footer** — Logo, then link columns: Explore, Resources, Documentation, Company, Legal.

---

## Fonts and colors

### Fonts

- **Primary:** [Inter](https://fonts.google.com/specimen/Inter) (Google Fonts) — weights **400**, **500**, **600**, **700**.
- **Fallbacks:** `-apple-system`, `BlinkMacSystemFont`, `'Segoe UI'`, `sans-serif`.

### Colors

| Use | Color | Notes |
|-----|--------|--------|
| Body text | `#1f2937` | Main content |
| Page background | `#fff` | White |
| Primary headings / dark UI | `#111827` | Titles, primary buttons |
| Secondary headings / medium text | `#374151` | Nav links, secondary buttons |
| Muted text | `#4b5563` | Descriptions, body secondary |
| Gray text / labels | `#6b7280` | Subtitles, footer labels, small caps |
| Placeholder / light gray | `#9ca3af` | Input placeholder, hover borders |
| Primary button (solid) | `#111827` | “Start for free”, “Start now”, “Get started”, etc. |
| Primary button hover | `#1f2937` | Darker on hover |
| Accent / teal (NEW badge) | `#0f766e` | “NEW” pill background |
| Accent / green (labels, links) | `#059669` | “ASSISTANT”, “ENTERPRISE”, footer link hover, pricing/quickstart links |
| Card / light background | `#f9fafb` | Feature cards |
| Borders / dividers | `#e5e7eb`, `#d1d5db` | Cards, inputs, nav border |
| Nav bar | `rgba(255, 255, 255, 0.95)` | Slight transparency |
| Nav border | `rgba(0, 0, 0, 0.06)` | Subtle bottom border |
| Shadows | `rgba(0, 0, 0, 0.06)`, `rgba(0, 0, 0, 0.08)` | Images and cards |
